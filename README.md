# Donation Management System

## Overview

The Donation Management System is a responsive web-based application developed using HTML, CSS, and JavaScript. The system is designed to assist non-profit organizations and charities in promoting donation campaigns, collecting donations, and interacting with potential donors through a simple and user-friendly interface.

This project provides essential pages such as Home, About, Campaigns, Donate, and Contact, making it suitable for educational purposes and frontend web development practice.

---

## Features

* Responsive and modern user interface
* Home page with introductory content
* About page describing the organization's mission and vision
* Campaigns section to showcase fundraising campaigns
* Donation page for donor interaction
* Contact page for communication
* User-friendly navigation
* Mobile and desktop compatible design

## Installation

To run this project locally, follow these steps:

1. Download or clone the repository.

```bash
git clone https://github.com/yourusername/donation-management-system.git
```

2. Navigate to the project directory.

```bash
cd donation-management-system
```

3. Open the `we.html` file in any web browser.

No additional dependencies or installations are required.
